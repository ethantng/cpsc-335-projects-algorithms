Overview:
This program is intended to solve a shortest-path problem with the requirement that there is a maximum number of stops which is the variable k. The program's goal is to find the cheapest route with at most k stops from the starting city to the destination.

How to Execute:
ensure g++ is installed
navigate to the directory the source file is in
ensure the txt files are in the same directory
compile using this command:
g++ -o CPSC362-Proj3-Algo2-EthanNguyen CPSC362-Proj3-Algo2-EthanNguyen.cpp
run the compiled program
./CPSC362-Proj3-Algo2-EthanNguyen
